# Backend Guidelines

Use this file for backend-specific rules. Keep backend details here instead of the root file whenever possible.

## Scope
- Spring Boot `3.5.9`, Java `21`, Maven, Spring AI `1.1.0`, LangGraph4j `1.8.8`
- Main code lives in `backend/src/main/java`
- Config lives in `backend/src/main/resources`
- Tests live in `backend/src/test/java`

## Backend Conventions
- Use tabs in Java files.
- Use `PascalCase` for classes.
- Keep packages under `com.silkycoders1.jsystemssilkycodders1`.
- Add backend-specific guidance here as it is learned instead of growing the root `AGENTS.md`.
